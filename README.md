Web Capstone Project

This workout website is designed to make training simple, organized, and effective for users of all fitness levels. Exercises are grouped by major muscle categories, including biceps, triceps, legs, shoulders, back, core, and cardio, making it easy to find exactly what you want to work on. Each section offers a variety of workout ideas so users can mix things up, avoid boredom, and target different muscle groups throughout the week. Whether someone wants a focused arm day, a full lower-body session, or a heart-pumping cardio workout, the site provides clear options to match their goals.  I will provide instructions for each exercise as well as instructional videos to help the user fully understand what they need to do for that exercise. I also want to include some suggested workouts for users who don’t really know what works best together or don’t want to make their own workout and would prefer to use a predesigned workout.

The name of the website is Workout Planner but that may change in the future.  One of the standout features of the website is the personal workout builder. Users can select specific exercises from any category and add them to their own custom workout plan, allowing for a fully personalized training experience. This makes it easy to create routines that fit individual preferences, time limits, or fitness goals, all in one place. By combining structured exercise categories with customization, the website helps users stay motivated, organized, and consistent with their workouts.
